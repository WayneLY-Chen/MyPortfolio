# Wayne's Portfolio & Editorial Studio 🚀

這不僅僅是一個個人網站，而是一個結合了 **AI 深度整合**、**即時通訊**、以及**極致視覺動畫**的個人作品集展示平台。專案採用現代化的 **React + Node.js + PostgreSQL** 架構，旨在展示極致的 UI/UX 設計與穩定、可擴展的後端整合能力。

## 🌟 核心特色 (Key Features)

### 🤖 Wobot AI 創意助理
- **智能對話與 RAG**：整合 Google Gemini API，並透過向量搜尋技術讓助理精準掌握作品集資訊。
- **感官動態反饋**：支援 Edge TTS 語音輸出、STT 語音辨識，並具備情境感應的表情動畫系統（會說話、會大笑、會眨眼）。
- **視覺創作**：整合 Stability AI 實現即時畫像生成。

### 🎨 極致視覺藝術
- **電影級動畫**：使用 **GSAP (ScrollTrigger, SplitType)** 打造無縫捲動體驗，結合 **Lenis** 實現流暢物理滑動感。
- **現代美學 UI**：運用玻璃擬態 (Glassmorphism)、自定義動態游標系統，提供沉浸式的操作體驗。

### 🎮 即時互動遊戲系統
- **多人群戰 (Boss Raid)**：透過 **Socket.io** 實現即時同步的魔王對戰邏輯與全域傷害排行榜。
- **經典遊戲重構**：內建精心優化的 2048、貪食蛇、小恐龍跑酷，展現純粹的邏輯控制實力。

### ✍️ 技術與設計部落格
- **效能優先**：具備前端 Cache 層優化與 AI 主題封面生成，完美平衡內容深度與載入效能。

---

## 🛠 技術棧 (Technical Stack)

- **前端**: `React 18`, `Vite`, `GSAP`, `Zustand`, `Tailwind CSS`, `Framer Motion`
- **後端**: `Node.js`, `Express`, `Socket.io`, `Passport.js (OAuth 2.0)`
- **資料庫/AI**: `PostgreSQL (Supabase)`, `pgvector`, `Gemini API`, `Stability AI`
